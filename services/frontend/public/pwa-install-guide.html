üõ†Ô∏è Troubleshooting Common Issues

Issue: Install Button Not Appearing

Possible causes:

  • Not served over HTTPS (required for PWA)
  • Manifest file not properly linked
  • Service worker registration failed
  • Missing required manifest fields
  • User previously dismissed install prompt

Solution: Check Chrome DevTools ‚Üí Application ‚Üí Manifest for errors

Issue: Icons Not Showing Correctly

Common problems:

  • Incorrect file paths in manifest
  • Missing icon sizes
  • Icons not square (must be 1:1 ratio)
  • File format not supported (use PNG)

Solution: Validate all icon paths and ensure all required sizes exist

Issue: Service Worker Not Updating

Cache invalidation strategies:

  • Update CACHE_NAME version in service worker
  • Use skipWaiting() to force immediate activation
  • Clear old caches in activate event
  • Implement update notification UI for users
